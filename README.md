# 🚀 AI Resume Analyzer (ML-Based)

An AI/ML-powered Resume Analyzer that evaluates resumes using semantic similarity and NLP.

## 🔥 Features

* 📄 Resume Parsing (PDF)
* 🧠 Semantic Matching using Sentence Transformers
* 🏷 Skill Extraction using NLP
* 📊 Match Score Calculation

## 🛠 Tech Stack

* Python (Flask)
* Sentence Transformers
* spaCy
* JavaScript (Frontend)

## 🚀 How to Run

### Backend

pip install -r requirements.txt
python app.py

### Frontend

Open index.html in browser
